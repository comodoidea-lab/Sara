「管理しない」という新しいタスク管理体験を提供する、神秘的なタスク管理アプリ**Sara（さら）**を構築しました。
このアプリは、ユーザーがタスクを管理するのではなく、タスクが宇宙の周期のように自然に現れ、消えていく体験を重視しています。
実装した主な機能とこだわり

Celestial Arcana（天のアルカナ）デザイン:
深紺の夜空（
#0d1228）を背景に、金縁のカードが浮かび上がる神秘的なビジュアルを採用。
Noto Serif JP（見出し）とSpace Grotesk（ラベル）の組み合わせにより、占いのような凛とした空気を演出しました。
「管理しない」ための独自ルールを自動化:
今日の3枚: 1日のタスクは最大3つまで。完了したタスクは背景に沈み、新しいタスクが手前にせり出す「Timescape風」のカードスタックを実装。
夜のリセット: useAppStateフックにより、日付が変わると「今日」のタスクは完了・未完了を問わず自動的にクリアされます。
ストックの浄化: ストックされたタスクは3日経過すると自動的に消滅します。
一期一会の提案: ストック全体は見えず、現在の候補1つだけがランダムに提案されます。
情緒的なアニメーション:
完了の儀式: 完了ボタンを押すと、カードが金箔で修復されたような（金継ぎ風）テクスチャに変化し、奥へと消えていくアニメーションをframer-motionで実装しました。 プロジェクト構成

src/hooks/useAppState.ts: 日次リセットや期限切れロジック、LocalStorageへの保存を統括するコアロジック。
src/components/ArcanaCard.tsx: タロットカードのような質感と、深みのあるスタックアニメーションを実現。
src/index.css: Tailwind V4を活用した、Celestial Design専用のテーマ定義。 ユーザーは「今日、これだけはやろう」と選んだ3つのアルカナに向き合うだけでよく、過ぎ去ったタスクや積み上がったストックに追われることはもうありません。
